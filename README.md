# Simansaysgame
A fast and fun memory game inspired by the classic Simon Says. Watch the sequence of colors, repeat it in order, and see how far you can go! Each round gets harder — one mistake ends the game.
#Features
-Classic Simon Says–style memory gameplay

-Color-based sequence generation

-Smooth button animations

-Increasing difficulty as the pattern grows

-Game-over detection on incorrect input

-Restart button to begin a new game

-Simple, clean, and responsive design
